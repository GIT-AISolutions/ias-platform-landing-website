# Claude brief

This folder contains a compact copy of the OpenCodie homepage only.

Goal:
- Add more high-end movement and scroll animation.
- Keep the current interface, layout, content, colors, and brand style mostly intact.
- Improve motion quality, not redesign the full page.

Good directions:
- Smooth section reveal animations while scrolling.
- Subtle parallax or depth in the hero/dashboard area.
- Premium-feeling hover states and micro-interactions.
- Scroll-triggered movement for cards, pricing blocks, FAQ rows, and logo marquee.
- Keep performance reasonable and respect `prefers-reduced-motion`.

Files:
- `index.html`: homepage markup.
- `css/styles.css`: all homepage styling.
- `js/main.js`: homepage interactions and animation logic.
- `js/chatbot.js`: floating chatbot widget.
- `images/`: only the images used by the homepage.

Please do not convert this into a different UI. Treat it as an animation and polish pass.
